✨ Aprende Kiro

Tu IDE con IA para construir más rápido

¿Qué es Kiro?

Kiro es un IDE potenciado por IA diseñado para desarrolladores. Combina un editor de código inteligente con un agente de IA que entiende tu proyecto, genera código, escribe tests y te ayuda a iterar más rápido que nunca.

¿Cómo funciona?

  • Specs — Define requisitos en lenguaje natural y Kiro genera el plan de implementación.
  • Hooks — Automatiza acciones del agente cuando ocurren eventos en el IDE.
  • Steering — Guía al agente con reglas y contexto específico de tu proyecto.
  • Autopilot — Deja que Kiro implemente cambios de forma autónoma mientras tú supervisas.

⚡ Vibe Coding

Crea una app desde cero

Describe lo que quieres construir en lenguaje natural. Kiro generará toda la estructura, el código y los archivos necesarios automáticamente.

Crea una app simple para subir documentos PDF y generar resúmenes automáticos con IA. Debe tener una interfaz web con drag & drop, mostrar el resumen en pantalla y permitir descargarlo como texto.
1
Abre Kiro y crea una carpeta de proyecto vacía
2
Abre el chat de Kiro (panel lateral)
3
Pega el prompt y pulsa Enter
4
Observa cómo Kiro genera los archivos en tiempo real
📋 Spec

Añade una feature con Spec

Los Specs permiten planificar e implementar features de forma estructurada. Kiro genera requisitos, diseño y tareas antes de escribir una sola línea de código.

Usando Kiro Specs, añade autenticación de usuarios a la app existente. Necesito registro con email/contraseña, login, logout y que solo los usuarios autenticados puedan subir documentos.
1
En Kiro, abre el panel de Specs (icono 📋 en la barra lateral)
2
Crea un nuevo Spec y pega el prompt
3
Revisa los requisitos que Kiro genera y apruébalos
4
Activa Autopilot y deja que Kiro implemente las tareas

💡 Tip: Si Kiro no se abre automáticamente, descárgalo en kiro.dev e instálalo primero.
